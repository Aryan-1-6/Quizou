Requirements :

-install Django
https://docs.djangoproject.com/en/4.2/topics/install/

-install Anaconda (optional) 
https://docs.anaconda.com/free/anaconda/install/

-install crispy-forms templates via terminal
pip install django-crispy-forms


How to operate :

Save Quizou folder to c:/users/your_directory/

IN CMD:

-navigate to Quizou directory (cd Quizou)
		
-*Run the following commands;
	python manage.py makemigrations
	python manage.py migrate
	python manage.py runserver

-hover over url and press [ctrl + click]

-Timed Quizzes

-*Objective Type*
